# 嘉亨物流官网

深圳嘉亨物流有限公司官方网站，展示公司核心业务、运营流程、管理制度、应急预案及服务质量承诺。

## 在线访问

**已部署地址：** https://jiaheng-logistics-website.vercel.app

## 本地运行

**环境要求：** Node.js

```bash
# 安装依赖
npm install

# 启动本地开发服务器（默认端口 3000）
npm run dev
```

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动本地开发服务器 |
| `npm run build` | 构建生产环境代码 |
| `npm run lint` | 运行 TypeScript 类型检查 |
| `npm run clean` | 清除构建产物 |

## 部署

本项目基于 [Vite](https://vitejs.dev/) + [React](https://react.dev/) + [Tailwind CSS](https://tailwindcss.com/) 构建，代码推送到 GitHub 后自动同步至 [Vercel](https://vercel.com/) 进行部署。
